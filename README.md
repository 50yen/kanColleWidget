# 開発

- nodejs: 7.0.0
- ~~npm: 3.8.9~~
- yarn: 0.19.1

```sh
git clone git@github.com:otiai10/kanColleWidget.git
cd kanColleWidget
yarn install && yarn start
```

- `yarn run build` きれいなjsをビルドするやつ
- `yarn start` 編集差分を見てビルドするやつ
- `yarn release` **release/kcwidget** にリリースビルドをつくるやつ
